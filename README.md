# PPY 的个人技术博客

一个现代化的个人技术博客网站，采用响应式设计，支持中英文切换和暗黑模式。

## 在线预览

访问地址：[https://paangbaobao.cn](https://paangbaobao.cn)

## 主要特点

### 1. 视觉设计
- 现代简约的界面设计
- 流畅的动画效果
- 星空背景与流星效果
- 3D 交互式头像
- 自适应的暗黑模式

### 2. 交互功能
- 中英文语言切换
- 明暗主题切换
- 平滑滚动
- 技能卡片悬停效果
- 社交媒体快速链接
- 邮箱一键复制

### 3. 技术特性
- 响应式布局设计
- CSS Grid 和 Flexbox 布局
- CSS 自定义属性
- 原生 JavaScript 实现
- 本地化存储偏好设置

## 技术栈

### 前端技术
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- CSS 动画和过渡效果
- CSS 变量
- LocalStorage API

### 第三方资源
- Font Awesome (图标)
- Google Fonts (Roboto)
- DevIcon (技能图标)
